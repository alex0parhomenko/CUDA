В проекте использована библиотека opencv для обработки изображений.
Для сборки надо запустить make
Чтобы запустить программу нужно набрать 
./solution test/390469629164.jpg 4 
где test/390469629164.jpg - имя изображения к которому будет применен медианный фильтр, 4 - радиус для фильтра
результат обработки будет лежать в ./result.jpg
