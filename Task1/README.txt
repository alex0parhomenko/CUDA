Сборка:
make

Запуск:
Сначала надо сгенерировать матрицу, для этого запускаем 
./gen_matrix.py -s 300
У нас появляется два файла matrix.txt там лежит матрица вместе с правой частью, дописанной справа к матрице.
Далее запускаем наше решение 
./solution matrix.txt > answer.solution.txt
Далее проверяем правильно ли наше решение нашло решение
./check_answer.py -s 300 -f1 answer.txt -f2 answer.solution.txt
